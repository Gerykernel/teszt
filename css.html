Formázás és Elrendezés

CSS3 stíluslapok A-Z-ig

A CSS határozza meg a HTML elemek vizuális megjelenését, elrendezését, és felelős a reszponzív (mobilbarát) működésért is.

Szelektorok és Prioritás

Az elemeket osztályok (`class`), egyedi azonosítók (`id`) vagy maguk a tagek alapján formázhatjuk:

/* Class szelektor */
.nav-link { color: #94a3b8; }

/* ID szelektor */
#navIsland { background: #000; }

/* Összetett szabályok */
.nav-link.active { color: #38bdf8; }

A Doboz Modell

Minden vizuális elem méretét négy réteg határozza meg:

  • Content: A tényleges tartalom mérete.
  • Padding: Belső margó a tartalom és a keret között.
  • Border: Az elemet körülvevő keret vastagsága.
  • Margin: Külső margó más elemekhez képest.

Modern Elrendezési Technológiák

Felejtsd el a régi `float` és `table` alapú elrendezéseket. A modern frontend fejlesztés két pillére:

Flexbox (Flexible Box Layout): Egydimenziós elrendezésekhez (sorok vagy oszlopok). Kiválóan alkalmas elemek függőleges és vízszintes középre igazítására, valamint rugalmas elosztására. Ezt használtuk a Dynamic Island belsejében a linkek elosztásához.

CSS Grid Layout: Kétdimenziós rácsrendszer. Egyszerre kezeli a sorokat és az oszlopokat, így bonyolult magazin-szerű vagy dashboard elrendezések készíthetők vele minimális kóddal (mint a fenti kétoszlopos infóblokk).
