A CSS3 Mesterkurzus

Animációk, reszponzivitás és modern vizuális elemek.

1. Animációk és Átmenetek

A weboldalak interaktivitását a mozgások adják. Két fő módja van az animálásnak CSS-ben:

Transitions (Átmenetek)

A transition akkor lép életbe, ha egy elem állapota megváltozik (pl. rávisszük az egeret). A fenti Dynamic Island is ezt használja.

.button {
    background: blue;
    transition: background 0.3s ease;
}
.button:hover {
    background: red;
}

Keyframes (Kulcskeretek)

Komplex, több lépésből álló animációkhoz használjuk, amelyek maguktól is lefuthatnak.

@keyframes lebeges {
    0% { transform: translateY(0); }
    50% { transform: translateY(-10px); }
    100% { transform: translateY(0); }
}
.karty { animation: lebeges 2s infinite; }

2. Reszponzív Webdesign (Mobilnézet)

Ma már elengedhetetlen, hogy egy weboldal telefonon is jól nézzen ki. Ehhez a **Media Query**-ket használjuk.

/* Asztali nézet (alapértelmezett) */
.container {
    width: 800px;
}

/* Ha a képernyő szélessége kisebb, mint 768px (telefon/tablet) */
@media (max-width: 768px) {
    .container {
        width: 100%;
        padding: 10px;
    }
}

Pro tipp: Mindig "Mobile First" (mobilra tervezünk először) elv szerint érdemes dolgozni!

3. Pseudo-osztályok és Elemek

A pseudo-osztályok egy elem speciális állapotát írják le (anélkül, hogy a HTML-t módosítani kéne).

  • :hover - Amikor az egér az elem fölött van. Ezzel oldottuk meg a Dynamic Island nyílását!
  • :nth-child(odd) - Kiválasztja minden páratlan elemet (pl. táblázat sorainak színezésére hasznos).
  • ::before és ::after - Extra tartalmat tudsz "befűzni" egy elem elé vagy mögé, tisztán CSS-ből. Gyakran használják ikonokhoz vagy design díszítőelemekhez.
