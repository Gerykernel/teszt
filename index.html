Webfejlesztés A-Z-ig

Modern alapok: HTML5 struktúra és CSS3 formázás.

A HTML (HyperText Markup Language)

A HTML adja a weboldalak vázát. Nem programozási nyelv, hanem egy leíró nyelv, amely tagekkel (címkékkel) mondja meg a böngészőnek, hogy mi hol található.

1. Az alapstruktúra

Minden weboldal egy szabványos "csontvázzal" kezdődik:

<!DOCTYPE html>
<html lang="hu">
<head>
  <meta charset="UTF-8">
  <title>Oldal címe</title>
</head>
<body>
  <!-- Ide jön a tartalom -->
</body>
</html>

2. Szövegformázás és linkek

A leggyakrabban használt tartalmi elemek:

  • <h1> - <h6>: Címsorok
  • <p>: Bekezdés (Paragraph)
  • <a href="url">: Kattintható hivatkozás
  • <img src="kep.jpg" alt="leírás">: Kép beillesztése
  • <div> és <span>: Csoportosító konténerek

3. Modern (Szemantikus) HTML5 elemek

Ahelyett, hogy mindent <div>-ekkel oldanánk meg, a HTML5 bevezette a jelentéssel bíró elemeket, ami jobb a SEO-nak (Google) és a képernyőolvasóknak:

<header>, <nav> (menü), <main> (fő tartalom), <section> (szekció), <article> (cikk), <footer> (lábléc).

A CSS (Cascading Style Sheets)

A CSS felel a kinézetért. A színek, az animációk, és az olyan modern megoldások, mint ez az üveghatású kártya vagy a fenti Dynamic Island, mind CSS-ből épülnek fel.

1. Szelektorok (Hogyan fogjuk meg a HTML-t?)

/* Tag szelektor (Minden p elem) */
p { color: blue; }

/* Class (Osztály) szelektor (Ponttal kezdődik) */
.glass-card { border-radius: 20px; }

/* ID szelektor (Kettőskereszttel kezdődik) */
#dynamicIsland { background: black; }

2. A Doboz Modell (Box Model)

Minden elem egy doboz. Ennek részei:

  • Content: Maga a tartalom (szöveg, kép).
  • Padding: Belső margó (a tartalom és a keret között).
  • Border: A keret.
  • Margin: Külső margó (elemek közötti távolság).

3. Elrendezés (Flexbox & Grid)

A modern pozicionálás alapjai:

Flexbox: Egy dimenzióban (sor vagy oszlop) rendezi az elemeket. Tökéletes menüköz, középre igazításhoz.
display: flex; justify-content: center;

Grid: Két dimenzióban (sorok és oszlopok rácsa) dolgozik. Komplett weblap-elrendezésekhez.

4. Modern CSS extrák (Amiket itt is használtunk)

Backdrop Filter: Ez adja az "üveghatást" (Glassmorphism), lemossa a mögötte lévő hátteret: backdrop-filter: blur(16px);

Transitions: Sima, folyamatos átmenetek az animációkhoz: transition: all 0.5s ease;

Variables: CSS változók az egyszerű színkezeléshez: --bg-color: #0f172a;
